Requires python2.7 32bit and pygame

Run:
pythonw chickn.py

Controls:

 * "up" -> Jump
 * "down" -> Slam / Charge launch
 * "left" -> Move Left
 * "right" -> Move Right

WASD, IJKL, ArrowKeys -> "up left down right"

Gamepad:

 * LeftStick/Stick1 -> "left" & "right" (on x axis) 
 * A/1 -> "up"
 * B/2 -> "down"
